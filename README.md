# 碳足迹计算器 - 25-10矿大能动

本项目是一个面向中国矿业大学低碳能源动力工程专业25级10班学生的碳足迹计算网站，旨在帮助学生了解和管理个人碳足迹，提高低碳环保意识。

## 网站功能

### 学生用户版本
1. **碳足迹计算工具**：对比不同活动（如交通方式）的碳排放量差异
2. **个人碳足迹记录**：保存和查看历史计算记录
3. **统计分析**：展示个人碳足迹统计数据
4. **班级排名**：查看班级同学的碳足迹排名情况

### 管理员/教师版本
1. **数据管理后台**：查看所有用户的碳足迹数据
2. **班级整体统计**：分析班级整体碳足迹情况
3. **数据导出**：导出CSV格式的班级数据

## 使用说明

1. **访问方式**：直接在浏览器中打开 `index.html` 文件

2. **碳足迹计算**：
   - 进入"碳足迹计算"页面
   - 选择两种不同的交通方式
   - 输入距离（公里）
   - 点击"计算碳足迹"按钮查看结果
   - 可选择保存记录并添加备注

3. **查看记录**：进入"个人记录"页面查看历史计算记录

4. **查看排名**：进入"班级排名"页面查看班级同学的排名情况

5. **管理员登录**：
   - 进入"管理员登录"页面
   - 默认用户名：`admin`
   - 默认密码：`2510_admin`
   - 登录后可查看班级数据统计和导出数据

## 技术说明

- **前端技术**：HTML5, CSS3, JavaScript
- **数据存储**：使用浏览器本地存储（localStorage）
- **计算依据**：基于不同活动类型的碳排放系数进行计算
- **响应式设计**：适配不同屏幕尺寸的设备

## 碳排放系数说明

网站使用的碳排放系数基于公开的研究数据，主要包括：

- 交通方式：步行、自行车、公共汽车、地铁、出租车、私家车等
- 能源消耗：电力、天然气、煤炭、汽油、柴油等
- 饮食：各种食物类型的生产碳排放

## 注意事项

1. 本网站仅用于教育和宣传目的，计算结果仅供参考
2. 管理员密码应定期更换以保证数据安全
3. 本地存储的数据仅保存在当前浏览器中，清除浏览器数据会导致数据丢失
