# 🚀 碳足迹计算器 - 快速开始

## ⚡ 5秒开始使用

1. **确保服务器运行中**
   ```bash
   python -m http.server 8080
   ```

2. **打开浏览器访问**
   ```
   http://localhost:8080/calculator-fixed.html
   ```

3. **开始计算！**
   - 选择交通方式或食品类型
   - 输入距离或重量
   - 点击"开始计算"
   - 查看结果！
